# 2025-billionaires-interactive-dashboard
Interactive Excel dashboard analyzing the world’s richest people in 2025 – wealth, industries, self-made, gender gap &amp; more


**Fully interactive Excel dashboard** analyzing the world's **top 475 billionaires in 2025** with **combined wealth > $4.2 trillion** (top 10 alone own ~23%).

Live Excel File → [Download Here](https://github.com/marwa-hemdan/2025-billionaires-interactive-dashboard/blob/main/data/Billionaires%20Dashboard%20START(dashboard).xlsx)

## Key Insights (2025)
| Metric                        | Value 2025              | Notes                                      |
|-------------------------------|-------------------------|--------------------------------------------|
| Total Billionaires (top 475)  | 475                     |                                            |
| Total Wealth                  | **$4.2+ trillion**      | Top 10 = $1.05 trillion (23.4%)            |

| Average Age                   | 65.8 years              |                                            |

## Dashboard Features
- 4 interactive slicers (Region • Industry • Gender • Self-Made)
- 2 professional charts (Barchart, Line chart)
- One-click “Reset All” button
- Fully dynamic – no broken links
- Built 100% in Excel (no Power BI, no Python)

1. Download the Excel file
2. Open & enable macros (only used for Reset button)
3. Click any slicer to explore instantly!
